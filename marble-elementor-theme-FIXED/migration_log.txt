1. INSTALL REQUIRED PLUGINS:
   □ Elementor (free version or Pro)
   □ Contact Form 7 or WPForms (for contact page)
   □ Optional: Elementor Pro for advanced features

2. WORDPRESS CONFIGURATION:
   □ Go to Settings → Permalinks
   □ Select "Post name" permalink structure
   □ Click "Save Changes" (flushes rewrite rules)

3. UPLOAD MEDIA FILES:
   □ Check elementor-pages/*.html for image references
   □ Upload images to WordPress Media Library
   □ Images referenced: wp-content/uploads/2021/02/, wp-content/uploads/2020/10/, etc.
   □ Alternative: Place images in wp-content/uploads/ matching the paths in HTML

4. VERIFY PAGES IN ELEMENTOR:
   □ Edit each page with Elementor
   □ Click "Update" to save Elementor data
   □ This converts HTML content to Elementor-editable format

5. OPTIONAL CUSTOMIZATIONS:
   □ Add custom logo: Appearance → Customize → Site Identity
   □ Add location pages to footer menu if desired
   □ Customize colors and fonts in Elementor global settings
   □ Set up contact form on Contact Us page

DEACTIVATION:
-------------
To disable auto-page creation on re-activation:
- Comment out line in functions.php:
  // add_action( 'after_switch_theme', 'marble_elementor_theme_create_pages_on_activation' );

To rebuild menu manually:
- Call function: marble_elementor_theme_rebuild_menu()
