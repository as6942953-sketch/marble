=== Marble Elementor Theme ===

Theme Name: Marble Elementor Theme
Version: 1.0
Author: Nauman Ellahi
Description: Converted static HTML + Elementor markup to WordPress theme. Fully compatible with Elementor page builder for visual editing.

== Installation ==

1. Install the Elementor plugin from WordPress.org
2. Upload the theme zip file via WordPress Admin > Appearance > Themes > Add New > Upload Theme
3. Activate the theme
4. Go to Appearance > Menus to set up your navigation menu
5. Go to Templates > Theme Builder in Elementor to set up Header and Footer templates (if you have Elementor Pro)

== Requirements ==

- WordPress 5.0 or higher
- Elementor plugin (free version)
- Elementor Pro (recommended for Header/Footer templates)

== Features ==

- Full Elementor compatibility
- Responsive design
- Clean, minimal codebase
- Ready for visual editing with Elementor

== Theme Structure ==

- style.css - Main stylesheet with theme header
- functions.php - Theme functions and Elementor support
- header.php - Header template with Elementor location support
- footer.php - Footer template with Elementor location support
- index.php - Main template file
- page.php - Page template
- single.php - Single post template
- front-page.php - Front page template

== Support ==

For Elementor-specific questions, please refer to Elementor documentation.
For theme customization, edit the template files as needed.
